# LungSegmentation

Segmentation of lungs from CT images. Run the code by passing arguments to the main.py. The segmented lung and its masks are stored in the --data_output folder.


data folder should be organized as below;

```bash
DATA FOLDER                
 ├──  images
 |      ├──Patient_0
 |           ├── 1.DICOM     
 |           ├── ...
 |           └── n.DICOM
 |      ├──Patient_1
 |           ├── 1.DICOM     
 |           ├── ...
 |           └── n.DICOM
 |      ├──Patient_2
 |           ├── 1.DICOM     
 |           ├── ...
 |           └── n.DICOM

```
